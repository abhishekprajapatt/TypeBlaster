# TypeBlaster

A fast-paced, space-themed typing game inspired by ZType where you defend against enemy ships by typing words correctly.

![Screenshot 2025-05-20 120724](https://github.com/user-attachments/assets/7211f77e-d539-404f-8afe-da0a7b8d1b44)

## 🎮 Game Overview

TypeBlaster challenges your typing speed and accuracy as you defend against waves of enemy ships. Each ship displays a word - type it correctly to destroy the ship before it reaches your base!

## ✨ Features

- **Dynamic Difficulty**: Game gets progressively harder with each level
- **Responsive Design**: Plays well on mobile, tablet, laptop, and large desktop screens
- **Visual Effects**: Lasers, explosions, and visual feedback for every keystroke
- **Sound Effects**: Immersive audio for gameplay actions (can be muted)
- **Performance Tracking**: Score, accuracy, and level progression
- **Modern UI**: Clean, space-themed interface with animated elements

## 🚀 How to Play

1. Type the words shown on enemy ships
2. Each correct character fires a laser beam at the targeted ship
3. Complete the word to destroy the ship
4. Prevent ships from reaching your base
5. Progress through increasingly difficult levels

## 🔧 Technical Details

TypeBlaster is built with:
- HTML5
- CSS3 (with animations and responsive design)
- Vanilla JavaScript (no frameworks)

## 🌐 Browser Compatibility

- Chrome (recommended)
- Firefox
- Safari
- Edge

## 📱 Responsive Design

The game is fully responsive and optimized for:
- Small mobile devices (up to 480px)
- Medium mobile devices (481px - 767px)
- Tablets (768px - 1023px)
- Small laptops (1024px - 1365px)
- Large laptops and desktops (1366px - 1919px)
- Large desktops and 4K displays (1920px+)
